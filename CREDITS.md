## Tileset Art — Liberated Pixel Cup (LPC)

This project uses artwork from the **Liberated Pixel Cup** (LPC), a joint project between the Free Software Foundation and OpenGameArt.org.

### LPC Terrain Tileset

- **Source**: [OpenGameArt.org — LPC Terrain](https://opengameart.org/content/lpc-terrain-repack)
