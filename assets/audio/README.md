# Ambient audio (phase 2733)

CC0 procedural placeholders generated locally with FFmpeg (`sine` lavfi, mono PCM
16-bit, 44100 Hz, 1 s). They are mathematically seamless when looped end-to-end
(integer cycles at 44100 Hz).

Replace with CC0/CC-BY music from [Freesound](https://freesound.org) or
[OpenGameArt](https://opengameart.org) if you want production ambience; keep total
game audio under the HTML5 budget (~5 MB) as needed.
